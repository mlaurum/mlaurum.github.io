My Projects

Secure Dropbox

In this project I designed and implemented a secure file store reminiscent of how dropbox stores its files. This file store can be used to store personal files securely, or to share files with other people you trust. My security implementation has mainly two properties: Confidentiality. Any data placed in the file store should be available only to you and people you share the file with and Integrity: You should be able to detect if any of your files have been modified while stored on the server and reject them if they have been.



Pintos OS

For this project we designed a buffer cache integrated into the file system. Implemented extensible files using linus style inode data-structures. Created subdirectories for easier navigation of the file system








Working around NP problems

Project regarding figuring out how to deal with NP problems. Worked to code a solution to a more general version of the classic knapsack problem. I implemented and tested various different strategies including Integer linear programming and max flow. I used each models performance against a 5 gigabyte set of items in order judge how good they were.
